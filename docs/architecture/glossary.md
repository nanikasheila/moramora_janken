# 用語集

プロジェクト固有のドメイン用語を定義する。

## ドメイン用語

| 用語 | 定義 |
| --- | --- |
| モラモラはとこちゃん | NPC キャラクター。じゃんけんの対戦相手 |
| シャッフル | NPC の手がランダムに切り替わっている状態（ゲーム待機中） |
| VS 演出 | ユーザー選択後のブラックアウト + VS 表示のトランジション |
| リザルトバー | 画面下部の勝敗結果表示エリア |
| 表情レイヤー | キャラクターの表情差分を重ねて表示する仕組み |
| ステータスピル | 「シャッフル中」「判定中」「結果発表」を示すラベル |
| GameLogic | game-logic.js が公開する純粋関数のオブジェクト |
| MORAMORA_CONFIG | config.js が定義するゲーム設定のグローバルオブジェクト |

## 技術用語

| 用語 | 定義 |
| --- | --- |
| UMD パターン | Universal Module Definition。Node.js（CommonJS）とブラウザのグローバルスコープの両方で動作するモジュール定義パターン。game-logic.js に採用 |
| 純粋関数 | 同じ入力に対して常に同じ出力を返し、副作用を持たない関数。テスト容易性が高い |
| クロージャ変数 | 関数スコープ内に閉じ込められた変数。app.js の `currentHand`、`isLocked` 等がこれに該当 |
| 差し替えレイヤー | 画像要素を重ね合わせ、CSS で表示/非表示を切り替えることでキャラクターの手・表情を表現する手法 |
