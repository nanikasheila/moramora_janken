# moramora_janken

モラモラはとこちゃんとじゃんけんだ！

---

GitHubPagesで公開することを前提に静的サイトを作成します。
これは、「モラモラはとこちゃん」というキャラクターの女の子と**じゃんけん**ができるページです。

下記は簡単な仕様です。

- 'assets\base.png'は常に表示し、同じ位置に差分画像を重ねることで表情変化＆手（じゃんけん）の変化を表現します。
- 表情差分は、'assets\expression_**.png'という画像です。
- 手の差分は、baseがグー、'assets\Scissors.png'がチョキ、'assets\Paper.png'がパーです。
- ゲームがスタートすると、所定時間間隔で手と表情の差分が切り替わります。
- ユーザーは、グー・チョキ・パーの何れかを選択することで、自分の手を決定します。
- ユーザの手の決定により、差分のランダム表示が停止し、キャラクターの手が決定します。
- この結果により、ユーザーの勝ち・負け・引き分け、が決定します。
- ゲーム結果を画面に表示します。
- ゲーム結果がでると、もう一度ゲームにトライできます。

## 追加仕様・設定

- controlsセクションはキャラクター表示の下に常時配置されます（画面幅が広くても狭くても同じ順序）。
- キャラクターは16:9の表示枠でバストアップを大きく表示し、枠からはみ出すような見え方に調整しています。
- じゃんけんの手の切り替わり速度は `js/config.js` の `shuffleIntervalMs` で変更できます（デフォルト: 250ms）。
- 勝ち・負け・あいこのセリフは `js/config.js` の `messages.win` / `messages.lose` / `messages.draw` に複数パターンを設定できます。
- キャラクター（モラモラ）が勝利した場合の表情は常に `assets/base.png` を使用します（ユーザー勝利時は固定しません）。
- 表情差し替え用の画像パスは `js/config.js` の `expressions` に列挙してください。
- 結果表示までのディレイは `js/config.js` の `revealDelayMs` で調整できます（デフォルト: 900ms）。
- 揺れ演出の長さは `js/config.js` の `shakeDurationMs` で調整できます（デフォルト: 420ms）。ユーザーが手を決めた瞬間に画面全体が揺れます。
- 結果はゆっくり表示され、判定中の演出で焦らします。

## 動作

- `index.html` をブラウザで開くだけで遊べます（GitHub Pages でそのまま公開可能です）。
- 「もう一度あそぶ」ボタンでシャッフルが再開されます。
